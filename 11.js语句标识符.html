js语句标识符

break 用于跳出循环

catch 与try连用 在执行错误的时候 执行catch语句块

continue 跳出当前循环

do while 在条件语句为true的时候执行该语句块

for 在条件语句为true时 可以将代码块指定执行的次数

for in 用于遍历数组或者对象的属性（对数组或者对象的属性进行循环操作）

function 定义一个函数

if else 用于基于不同的条件来执行不同的动作

return 退出函数

throw 抛出错误

try 实现错误处理 与catch一同使用

var 声明一个变量 存在变量提升

while 当条件语句为true时，执行语句块
